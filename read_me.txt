В программе присутствуют 3 класса.
1. Person отвечает за запись о пользователе. Сохраняется в файл в виде json
    объекта. При запуске все объекты json преобразуются в объекты Person

2. PhoneBook основной класс для хранения объектов Person и работы с ними.
    Вывод всех записей. Добавление записи. Редактирование записи. Поиск записи по
    фамилии или по организации. При запуске программы происходит выгрузка json
    объектов в объекты Person и сохранение их в список.

3. FileController необязательный класс со статическими функциями сохранения и загрузки
    json файла. Функция считывания возвращает список объектов Person. В случае
    если файл отсутствует, возвращает пустой список.

Функционал программы реализован сугубо в консоли. Но оставлены возможности для работы
с другими интерфейсами.
